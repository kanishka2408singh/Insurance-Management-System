# Insurance Management System

## Overview

Insurance Management System is a console-based application developed in C++ using Object-Oriented Programming (OOP) concepts and File Handling. The project helps manage insurance-related records such as customers, insurance plans, payments, agents, nominees, and feedback through a menu-driven interface.

## Features

* Customer Management (Add, View, Search, Update, Delete)
* Insurance Plan Management
* Custom Insurance Plans
* Payment Tracking and Status Management
* Agent Management
* Nominee Management
* Feedback Management
* Report Generation
* Data Validation
* File Handling for Permanent Data Storage
* Menu-Driven User Interface

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* File Handling
* Turbo C++ 3.0
* DOSBox Emulator

## Modules

* Customer-Nominee Linking

* Feedback Management

## Key Concepts Implemented

* Classes and Objects
* File Streams
* Data Validation
* Menu-Driven Programming
* Modular Programming
* Record Management System

## Project Objective

The main objective of this project is to demonstrate the practical implementation of C++ programming concepts for managing insurance records efficiently while reducing manual work and improving data organization.
